## 术语定义
为便于阅读，本文档对以下术语进行统一定义：

- **祈愿**：官方游戏内获取角色和武器的行为，本文中统一使用通俗说法“抽卡”。
- **角色活动祈愿 / 角色活动祈愿-2**：官方角色卡池名称，本文统称为“角色卡池”。
- **武器活动祈愿**：官方武器卡池名称，本文称为“武器卡池”。
- **神铸定轨**：官方武器卡池的定向选择机制，本文简称为“定轨”。
  - **命定值**：定轨机制中的计数，达到1点后下次五星必为定轨武器。
- **纠缠之缘**：官方抽卡道具，本文直接用“抽卡”或“抽数”表示一次消耗。
- **无主的星辉 / 无主的星尘**：官方名称，本文简称为“星辉”和“星尘”。
- **无主的命星**：官方名称，本文简称为“命星”，用于满命溢出后的特殊转化。
- **保底机制**：
  - **小保底**：出五星时，有50%概率为当期UP角色的情况。
  - **大保底**：上一次小保底未获得UP角色后，下一次五星必定为当期UP角色。
  - **捕获明光**：在小保底触发时，有极低概率直接获得UP角色的特殊机制。
- **抽数**：单次抽卡的单位，即一次抽卡。

## 项目概述
本项目旨在开发一套基于官方规则、融入社区智慧的原神抽卡模拟与分析系统，专注于 **6.4下半** 的限定角色卡池与武器卡池。它不仅能够真实还原抽卡过程，还能提供目标预设与进度追踪、武器卡池复杂策略引导、事前成功率预测与事后实际达成分析，帮助玩家科学规划抽卡资源，深入理解游戏机制。

## 📌 一、设计目标

### 1.1 核心目标
- **精准模拟**：严格遵循官方公示的抽卡规则，包括五星/四星概率、保底机制、捕获明光、武器卡池定轨、四星UP保底、重复角色转化规则等，并采用社区公认的软保底概率模型。
- **智能决策**：在武器卡池抽卡过程中，根据用户预设的目标和当前状态，提供符合游戏机制的策略选项（如双限定武器的“取消定轨重新定轨”策略）。
- **目标预设与追踪**：允许用户在模拟前设定每个卡池的期望获取数量（如“丝柯克 7 个”、“苍耀 1 把”），模拟过程中实时检测并暂停询问。
- **事前预测**：基于社区统计的期望值数据，在用户输入目标后，立即给出达成目标的预计抽数区间。
- **事后分析**：模拟结束后，对比实际消耗抽数与社区期望值，分析偏差，并提供抽卡策略效率评估。

### 1.2 用户价值
- 为玩家提供零成本的抽卡策略测试环境，验证“N命M精”所需资源的合理性。
- 通过策略引导，帮助玩家理解武器卡池定轨规则和社区发现的高阶玩法。
- 用数据说话，让抽卡规划从“凭感觉”升级为“有依据”。

## ⚖️ 二、设计原则

### 2.1 规则准确性原则
- 所有概率数据均取自官方公示，软保底采用社区公认模型：
  - **角色卡池**：1-73抽0.6%，74-89抽每抽递增6%，90抽100%。
  - **武器卡池**：1-62抽0.7%，63-74抽每抽递增7%，75-79抽每抽递增3.5%，80抽100%。
- 捕获明光采用递进补偿模型：连续歪0次→50%UP，1次→55%，2次→75%，3次及以上→100%UP；触发后下一个五星仍为小保底。
- 武器卡池定轨值上限为1，命定值积累规则与官方完全一致。
- 四星UP保底：角色卡池出四星时50%为UP角色，否则下次四星必为UP；武器卡池出四星时75%为UP武器，否则下次四星必为UP。

### 2.2 重复角色转化规则
> **若获得重复角色**
> - **五星角色**：第2~7次获得相同5星角色时，每次将转化为1个对应角色的命星和10个星辉；第8次及之后获得，将转化为1个命星和25个星辉。
> - **四星角色**：第2~7次获得相同4星角色时，每次将转化为1个对应角色的命星和2个星辉；第8次及之后获得，将仅转化为5个星辉。

### 2.3 保底继承规则
- **角色卡池**：即「角色活动祈愿」与「角色活动祈愿-2」的保底次数和状态完全共享，与武器卡池、常驻卡池独立。
- **武器卡池**：即「武器活动祈愿」，其保底次数在内部继承，但定轨的命定值不继承（本期活动结束重置）。

### 2.4 社区共识原则
- 采用社区统计的期望值作为预测基准：
  - **单UP角色期望抽数**：约93.36抽
  - **大保底上限**：180抽
  - **75%成功率抽数**：约135抽
  - **90%成功率抽数**：约155抽
  - **双限定武器各抽一把**：期望抽数约158.5抽
  - **双限定武器满精炼**：期望抽数约692.95抽

### 2.5 数据完整性原则
- 每次抽卡的详细信息记录至Excel，汇总表包含预设目标、达成情况、实际与期望偏差、重复角色转化统计等。
- 策略决策日志单独记录，用于最终报告的策略效率评估。
- **新增**：限定物品每次获取的真实抽数（从上次获得该物品到本次的总抽数）单独记录，便于核对是否歪保底。

### 2.6 真实场景聚焦原则
- 仅模拟 **6.4下半** 的**限定角色卡池（C1/C2）** 和**限定武器卡池（W）**。
- 不涉及常驻卡池（奔行世间）和集录祈愿，因其规则与限定池差异较大且使用频率低。

### 4.2 捕获明光机制
- 小保底时，有0.018%概率触发捕获明光，直接获得当期UP角色。
- 连续歪次数影响小保底UP概率：0次→50%，1次→55%，2次→75%，3次及以上→100%。
- 触发捕获明光后，下一个五星仍为小保底。

### 4.3 武器卡池定轨（神铸定轨）机制
- 命定值上限为1。若未获得定轨武器，命定值+1；达到1后下一把五星必为定轨武器。
- 获得定轨武器后命定值重置为0。
- 切换或取消定轨会清空命定值。
- **优化交互**：执行`W 次数`未指定定轨时，自动沿用已有定轨；仅当无定轨且未指定时才提示用户确认。

### 4.4 四星保底机制
- **角色卡池**：出四星时，50%概率为当期UP角色，否则下次四星必为UP角色。
- **武器卡池**：出四星时，75%概率为当期UP武器，否则下次四星必为UP武器。

### 4.5 角色命座与重复角色转化规则
- **五星角色**：
  - 第1次获得：0命，获得角色本身。
  - 第2~7次获得：每次获得1个对应角色的命星 + 10星辉（累计可升至6命）。
  - 第8次及之后获得：每次获得1个命星 + 25星辉，**不计入命座**。
- **四星角色**：
  - 第1次获得：获得角色本身。
  - 第2~7次获得：每次获得1个对应角色的命星 + 2星辉。
  - 第8次及之后获得：每次获得5星辉，**不计入命座**。

### 4.6 武器精炼规则
- 获得第1把为精1，第5把为精5（满精）。第6把及以上仅转化为星辉（4星武器2星辉，5星武器10星辉），不计入精炼等级。

## 🧠 六、武器卡池复杂决策支持

### 6.1 决策触发条件
武器卡池每获得一个五星，立即暂停，显示当前状态并提供策略选项。

### 6.2 策略选项生成
根据用户预设目标和当前状态动态生成选项（如双限定策略、更换定轨、保持定轨等）。示例：
```
✨ 当前定轨：苍耀，命定值=1
已获武器：苍耀 0 把，香韵奏者 0 把
剩余抽数：XX

请选择下一步操作：
1-无定轨继续
2-停止本次抽卡
3-更换定轨后继续
4-保持定轨继续（不清除命定值）
5-执行策略：取消定轨并重新定轨苍耀（推荐）
```

### 6.3 策略日志记录
每次用户做出策略选择，记录到策略日志，格式如：
```
[抽卡时间] 决策类型：更换定轨（W1→W2）
[抽卡时间] 决策类型：执行双限定优化策略
[抽卡时间] 目标达成：苍耀 1/1，停止抽取

## 📝 十三、总结

本方案完整实现了以下核心功能：

1. **精准抽卡模拟**：基于官方规则和社区概率模型，还原真实抽卡体验。
2. **物品数量精确核对**：确保所有物品清单与官方一致。
3. **重复角色转化**：严格遵循官方描述，记录第2-7次与第8次及之后的星辉/命星转化。
4. **目标预设与预测**：用户可设定抽取目标，系统提供达成预计抽数，帮助合理规划。
5. **武器卡池复杂策略支持**：每次出金后暂停，根据目标动态生成策略选项，模拟社区高阶玩法。
6. **限定物品真实抽数**：记录每次获取的真实抽数，便于核对是否歪保底。
7. **目标达成自动暂停**：避免资源浪费，支持满命溢出场景。
8. **事后分析与策略评级**：对比实际消耗与期望值，评估策略执行效果。
9. **趣味运气评价**：三维加权评分，9级等级+趣味彩蛋，增加报告可读性。
10. **图表自适应与字体兼容**：五星TOP10图表自动调整尺寸，Word报告字体统一回退。

所有设计均基于官方规则和社区共识，确保模拟的真实性和指导价值。本方案专注于 **6.4下半** 的限定卡池，不涉及常驻卡池和集录祈愿，符合绝大多数玩家的实际需求。
