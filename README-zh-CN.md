## 📖 项目简介
一个基于官方规则和社区概率的个人研究用抽卡模拟器，专注于 **6.4 下半** 限定角色卡池（丝柯克、爱可菲）与武器卡池（苍耀、香韵奏者）。支持目标预设、武器池策略决策、重复角色转化统计，并生成详细的 Excel、图表和 Word 分析报告。

---

## ✨ 核心功能
- **精准抽卡模拟**：完全复现官方概率，软保底采用社区公认模型（角色池73抽后6%递增，武器池分段递增）。
- **目标预设与预测**：启动时可设定抽取目标（如“丝柯克7个”、“苍耀1把”），系统基于社区期望值给出达成预计抽数。
- **武器池策略决策**：每次出金后暂停，根据目标动态提供策略选项（包括双限定武器的“取消定轨重新定轨”优化策略）。
- **目标达成自动暂停**：达到预设目标后暂停并询问是否继续，避免资源浪费。
- **重复角色转化规则**：严格遵循官方描述，区分第2-7次与第8次及以上获得的星辉/命星转化，并在终端和报告中提示。
- **限定物品真实抽数**：Excel 新增「限定物品详情」工作表，Word 报告新增章节，展示每个限定角色/武器每次获取的真实抽数（按获取顺序计算，每个物品独立统计保底间隔），便于核对是否歪保底。
- **武器池定轨交互优化**：未指定定轨时自动沿用已有定轨，仅当无定轨且未指定时才提示确认。
- **图表自适应**：五星物品 TOP10 图表根据物品数量自动调整尺寸，四星 TOP10 图表高度自适应，角色/武器五星饼图仅显示实际获得的类别。
- **Word 报告字体兼容**：优先使用微软雅黑，若系统缺失则自动回退到宋体，确保加粗效果统一。
- **四星物品星辉显示修复**：统一使用物品名称判断角色/武器，确保所有四星武器正常获得星辉并显示。
- **性能优化**：支持快速导出模式（选项5），大数据量（万级抽卡）下导出速度提升数倍；可选依赖 `xlsxwriter` 可显著提高 Excel 写入性能。
- **深度数据分析**：生成 Excel 记录、四张独立 PNG 图表和 Word 报告，包含基础统计、关键指标、目标达成分析、策略执行路径、运气评级与趣味彩蛋。
- **速度模式选择**：支持“真实节奏”（每10抽停顿1秒）和“快速模式”（无停顿），满足不同场景需求。
- **终端彩色输出**：五星金色加粗，四星紫色加粗，溢出物品特殊标记。
- **多语言支持**：通过 i18n 模块管理文本，支持中英文切换。

## 🚀 快速开始
1. **安装 Python 3.9+** 并确保已添加到 PATH。
2. **克隆或下载本项目**：
   ```bash
   git clone https://github.com/mypanda114/GenshinWishSimulator
   cd GenshinWishSimulator
   ```
3. **创建并激活虚拟环境**（推荐）：
   ```bash
   python -m venv venv
   .\venv\Scripts\activate   # Windows
   ```
4. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
   （可选）为获得更快的 Excel 导出速度，可额外安装 `xlsxwriter`：
   ```bash
   pip install xlsxwriter
   ```
5. **运行模拟器**：
   ```bash
   python src/genshin_wish_simulator.py
   ```

## 📤 导出选项
抽卡结束后，可选择：
1. **仅保存 Excel 记录**（角色记录、武器记录、限定物品详情、汇总表、策略日志）
2. **生成 Plotly 图表**（四张独立 PNG 图片：角色五星饼图、武器五星饼图、四星TOP10、五星TOP10）
3. **生成 Word 报告**（含所有 KPI、目标分析、策略路径、运气评级、限定物品详情）
4. **全部导出**
5. **快速导出**（仅保存 Excel 记录，无样式、无表格，适合大数据量，速度极快）

> **性能提示**：
> - 默认导出模式（选项1-4）会为 Excel 添加样式和表格，万级数据约需 30 秒～2 分钟。
> - 使用快速导出模式（选项5）时，导出时间可缩短至 5～10 秒，适合大量模拟（如 10000 抽以上）。
> - 推荐安装 `xlsxwriter` 以获得最佳写入性能（安装后自动使用）。

所有文件自动保存在 `output/原神抽卡模拟_YYYYMMDD_HHMMSS/` 目录下。

## 🌍 多语言支持说明
模拟器支持中英文切换，所有提示文本通过 `i18n.py` 管理。默认语言为中文，如需切换英文，请修改 `src/genshin_wish_simulator.py` 中的初始化语句：
```python
